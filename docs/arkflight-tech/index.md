# Arkflight Technology

Arkflight is made possible by a blend of magic, engineering, faith, risk, and ancient materials.

## Core Technologies

- **Arkengines** - reactor-engines that allow void travel.
- **Aetherite** - black oil-slick mineral used in cores and advanced magical systems.
- **Levstone** - unstable lifting stone used in elevators, ship systems, and cargo work.
- **Ark Compass** - navigation instrument that reads resonance, routes, currents, and jump points instead of north.

This section is player-facing. Secret truths about ancient engines, living aetherite, or hidden planetary systems belong in the GM guide.
